# HTTP Boot Resources

This directory contains bootable images and resources served over HTTP.

## Directory Structure:
```bash
/srv/http/
├── ubuntu-20.04/
│   ├── vmlinuz
│   ├── initrd
│   └── preseed.cfg
├── ubuntu-22.04/
│   ├── vmlinuz
│   ├── initrd
│   └── preseed.cfg
└── ubuntu-24.04/
    ├── vmlinuz
    ├── initrd
    └── preseed.cfg
```
